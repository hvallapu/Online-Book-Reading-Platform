# Online-Book-Reading-Platform
The Online Book Store is an E-learning platform that provides customers to read any book online by creating an account in the application.  In this system, the developer provides an account to every user and a limited book stock to every user. Here the developer may also apply a payment gateway to earn some money by deciding the price for every book. Here we can also provide other functionality like the translation of the book into another language.
